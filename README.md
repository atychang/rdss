# RDSS
Applying for the quota of RDSS, write in Python3

## Install
``` bash
$ pip3 install -r requirements.txt
```

## Configuation
Open `rdss.py` and replace line 7 to 9 with your configuation
``` python
# example
USERNAME = 'username'
PASSWORD = 'password'
BATCH = '70' # batch number
```

## Usage
``` bash
$ python3 rdss.py
```
